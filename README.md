# Sistema de Torneos de Fútbol

Proyecto desarrollado con Flask y SQLite para la gestión de torneos de fútbol.

## Funcionalidades
- Registro de equipos
- Administración de torneos
- CRUD completo

## Tecnologías
- Python (Flask)
- SQLite
- HTML5
- CSS3
